Detailed Description

The Session tables. Formally defined in RFC 6146 section 3.2.

bool nat64_add_session_entry ( struct session_entry *  entry)

Adds "entry" to the session table whose layer-4 protocol is "entry->protocol". Expects all fields but the list_heads from "entry" to have been initialized.

Because never in this project is required otherwise, assumes the entry is not yet on the table.
