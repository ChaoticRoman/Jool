Returns the BIB entry you'd expect from the "tuple" tuple.

That is, when we're translating from IPv6 to IPv4, returns the BIB whose IPv6 address is "tuple"'s source address. When we're translating from IPv4 to IPv6, returns the BIB whose IPv4 address is "tuple"'s destination address.
